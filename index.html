Cara Order via WhatsApp

Simple, cepat, dan tanpa ribet.

  1. Pilih Bouquet

    Buka halaman Koleksi lalu klik produk yang kamu suka untuk masuk ke halaman detail.

  2. Pilih Ukuran & Add-on

    Tentukan size M/L/XL, add-on, warna wrap, dan tulis catatan khusus bila ada.

  3. Tekan Tombol Order WA

    Sistem otomatis menyusun pesan agar admin langsung paham request kamu.

  4. Konfirmasi Detail

    Admin akan mengonfirmasi harga final, estimasi proses, dan info pengiriman.

  5. Order Diproses

    Setelah konfirmasi lengkap, bouquet dirangkai dan disiapkan untuk pengiriman.

FAQ

Pertanyaan yang paling sering ditanyakan customer AIRYBLOOM.CO.

Apakah bisa request warna tertentu?

Bisa. Tulis di catatan order atau langsung chat admin untuk request palette spesifik.

Apakah bisa same day delivery?

Bisa di beberapa area. Ketersediaan mengikuti slot produksi harian dan operasional kurir.

Kalau mau kirim sebagai surprise bisa?

Bisa banget. Sertakan nama penerima, nomor aktif, alamat, dan waktu kirim terbaik.

Apakah buket bisa custom budget?

Bisa. Diskusikan budget range melalui WhatsApp untuk rekomendasi paling pas.
